9月1号准备的面试题目：
1、SQL是什么？  结构化查询语言   与关系型数据库进行交流的语言。三大主流数据库有哪些？ mysql   orical   sql server 
2、当用户进行访问数据库的时候，他们是怎么进行交互的？ 用SQL语句
3、知识点：1、源代码不能当做数据 2、数据库需要安全和效率 3、SQL是structured (结构化) QUERY（查询） Language(语言)一种原因，是进行和数据库交流的工具
三大关系型数据库：oracle:也称为：PL/SQL  语法丰富、功能强大 适用各大中小微型机    MYSQL：也称为mysql    SQLServer:也称为T/SQL 可视化操作界面强大
当建立数据库的时候，会有mdf和mlf两个文件生成   mdf：存储用户录入数据库的数据   ldf:存储操作记录，也称为日志 可以利用ldf来进行备份 
-4-、SQL语句：SQL语句不区分大小写，自动忽略换行、注释：--两个短横线之后的内容都不会被执行。
-- 将数据库备份到C盘17bang.bak                               注意：不能在当前数据库上恢复当前数据库（需要切换数据库） 
  BACKUP DATABASE [17bang] TO DISK = 'C:\\17bang.bak'             需要指定备份文件名，不能是文件夹 
   -- 根据C盘17bang.bak恢复数据库                                  文件后缀名可以自由指定，.bak是约定俗成 
      RESTORE DATABASE [17bang] FROM DISK = 'C:\\17bang.bak'      确保指定文件路径有访问权限

 2、怎样去理解批处理？ 就是处理命令按一批来处理，有一定的先后顺序，这过程中会产生等待处理结果，也可以在第一局命令行之后加个GO，客户端会以go为分隔符，将命令分成若干批，分批处理。
 3、如何理解使用SQL时用go。
 4、数据库我们一般常用的有哪些类型： 文本数据库、数值、（整数、小数（decimal/numeric、float、real））、null值（空）、日期类型（date、time、datetime）、真假（1/0）
 5、nvarchar和varchar有什么区别？
 6、知识点复习：1、批处理：2，同义。   2、删除表：drop table+用户名  3、修改表结构：一定要先加 alter table+用户名  ADD Score DECIMAL(3,1);（增加列的意思） DROP COLUMN Score;（删除列的意思）
 7、ALTER COLUMN SCORE FLOAT NULL;（修改列的意思） 设置数据类型是为了：数据识别、预算、检验和高效存储的需要。 
 8、NVARCHAR 是使用Unicode编码的，使用时后面更一个（），里面的数值是数据的最长长度，不会补足。 Nchar是使用Unicode编码的，输入的数据长度是不变的，定长文本数据，始终为10，会补足，位数不够的以0来凑。
 9、VARchar和CHAR 都是使用非Unicode编码的，存储变长和定长的数据
 10、TEXT和NTEXT：不限长度的非UNICODE和unicode编码的文本数据。  凡是N开头的都是用unicode编码的，Unicode字符是一个字符占两个字节， 非unicode字符是一个字符占一个字节。
